## Dataset Construction

The full CSV contains ~14,640 tweets across three classes. The pipeline:

1. **Filter**: keep only `positive` and `negative` tweets (discard `neutral`)
2. **Balance**: sample exactly **500 positive + 500 negative** with `random_state=42`
3. **Result**: balanced dataset of **1,000 tweets** (50 / 50 split by class)

## Preprocessing Steps

1. **Lowercase** all text
2. **Remove URLs** (`http://...`, `www...`)
3. **Remove @mentions** (airline handles and user tags are noise)
4. **Expand hashtags** (`#great` → `great`)
5. **Remove HTML entities** (`&amp;`, `&lt;`, etc.)
6. **Remove punctuation** (keep only `[a-z\s]`)
7. **Stopwords**: kept by default (`remove_stopwords=False`) — negation words
   such as *not*, *never*, *no* are critical for sentiment and must be preserved
8. **Tokenize** with Keras `Tokenizer` (vocab size = 5,000, OOV token = `<OOV>`)
9. **Pad sequences** to length 50 (`padding="post"`, `truncating="post"`)
10. **Stratified 80/20 split** — preserves class balance in both sets
