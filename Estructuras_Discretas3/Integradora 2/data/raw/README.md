# Data Directory

## Dataset Source

*Name:* Sentiment Labelled Sentences Dataset  
*Source:* [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/Sentiment+Labelled+Sentences)  
*Citation:* Kotzias et al., "From Group to Individual Labels using Deep Features", KDD 2015.

## Raw Data (raw/)

Three .txt files, each with 1000 sentences (500 positive, 500 negative):

| File | Domain |
|------|--------|
| amazon_cells_labelled.txt | Amazon product reviews |
| imdb_labelled.txt | IMDB movie reviews |
| yelp_labelled.txt | Yelp restaurant reviews |

*Format:* <sentence>\t<label>\n — label is 0 (negative) or 1 (positive).

> The dataset is auto-downloaded when you first run any notebook. If it fails, download it manually from the UCI link above and place the three .txt files inside data/raw/.

## Processed Data (processed/)

Auto-generated by running notebooks/01_eda.ipynb and notebooks/03_dense_rnn_lstm.ipynb:

| File | Description |
|------|-------------|
| data_clean.csv | Cleaned and lowercased sentences with labels |
| X_train.npy | Training sequences (padded, shape: 2400×100) |
| X_test.npy | Test sequences (padded, shape: 600×100) |
| y_train.npy | Training labels |
| y_test.npy | Test labels |
| tokenizer.pkl | Fitted Keras Tokenizer |

> These files are excluded from version control (see .gitignore) since they can be regenerated.

## Preprocessing Steps

1. Lowercasing
2. Punctuation and non-alphabetic character removal
3. Stopword removal (NLTK English stopwords)
4. Tokenization (Keras Tokenizer, vocab size = 5000)
5. Sequence padding to max length 100 (padding="post", truncating="post")
6. Train/test split: 80% / 20%, stratified by label, random_state=42
